Privacy Policy

Last updated: August 2025

Docs Style Explorer is a Chrome extension designed to help users scan and bulk-format text styles in Google Docs. Your privacy is important to us, and this extension has been built with a “privacy-first” approach.

What the extension does

  • Scans the Google Docs document you explicitly open.
  • Detects text styles (headings, fonts, colors, lists, etc.).
  • Applies formatting changes that you choose (e.g., color, size, highlight).

Data collection

This extension does not collect, store, or share any personal data. All processing happens locally in your browser with your active Google Docs session. We do not run servers or transmit your content externally.

Permissions

The extension requests limited permissions to:

  • Access the current Google Docs file (to read and apply styles).
  • Use the Google Docs API with your permission via OAuth.
  • Store your extension settings (locally in your browser).

These permissions are used solely to provide the described functionality and nothing else.

Your control

You can revoke the extension’s access at any time via your Google account permissions (Google Account Settings) or by uninstalling the extension from Chrome.

Contact

If you have any questions or concerns about this privacy policy, please reach out via the Support page.
